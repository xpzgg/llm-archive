# 运算器设计

## 加法器

全加器（FA）：处理每一位的加法，产生本位和与进位。N 个全加器串联 = N 位加法器。

**关键问题**：进位必须从低位串行传到高位，这是加法延迟的来源（超前进位可优化）。

## 减法

没有减法器，复用加法器：

```
A - B = A + (-B) = A + (~B + 1)
```

实现：B 各位取反，加法器初始进位置1，自动完成。硬件上减法和加法共用同一个加法器。

## 比较运算

本质是做 `A - B`，看结果的符号位和是否为零，复用减法器，不需要新电路。

## 移位器

左移1位 = ×2，右移1位 = ÷2，直接挪位，不需要乘法器。

这就是为什么 `x << 1` 比 `x * 2` 快——移位是"免费"的。

## 整体规律

所有运算尽量复用加法器，加法器是 ALU 的核心，减法、比较都归结为加法，移位绕开加法器。
