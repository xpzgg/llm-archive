# 晶体管与逻辑电路

## MOS 晶体管

**本质**：电控开关，用 Gate 电场控制 Source-Drain 之间是否导通。

```
        Gate（栅极）—— 控制信号输入
          │
    ──────┴──────
    │   Metal   │
    │───────────│
    │   Oxide   │  ← 绝缘层，Gate 不流电流，只靠电场
    │───────────│
    │  Semicon  │
    ─────────────
   Source      Drain
```

**工作原理（NMOS）**：
- Gate 电压 < Vth：无导电沟道，断开
- Gate 电压 > Vth：电场吸引电子到表面形成 N 型沟道，导通

**关键设计**：Gate 与沟道之间隔着绝缘氧化层，输入端几乎不耗电，可以驱动大量后级电路。

## CMOS

**问题**：只用 NMOS + 电阻，稳定导通状态下电流从 VDD 持续流向 GND，白白耗散为热量。

**CMOS 解决方案**：PMOS（低电压导通）和 NMOS（高电压导通）互补串联：

```
VDD
 │
┤PMOS├  ← 输入高时截止
 │
 ├──── 输出
 │
┤NMOS├  ← 输入高时导通
 │
GND
```

**核心**：两者导通条件相反，永远一个通一个断，VDD 到 GND 之间永远有断点，静态功耗接近零。

功耗只在切换瞬间产生（两管短暂半导通），这就是为什么频率越高越耗电。

## 基本概念

- **VDD**：正电源（高电位，供电端）
- **GND**：地（低电位，电位为0，参考点）
- 电流从 VDD 流向 GND，中间电路消耗电位差做功

## 逻辑门到 CPU

```
晶体管 → 逻辑门（与/或/非）→ 加法器/寄存器 → CPU
```
